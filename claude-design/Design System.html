Design system

The component foundation for the Kraite operator console. Dark chrome, calm light content, terminal-green accent, monospaced numbers. Trading semantics are sacred — green is long / profit / safe, red is short / loss / risk, never inverted. Surfaces use a soft radius scale; numbers are the hero.

Foundations

Color

One signal color — terminal green — on a neutral ink scale. A fixed P&L pair and a five-step Black Swan regime palette carry all market meaning. No decorative gradients.

A soft, modern radius scale — distinct from the marketing site's hard edges. Surfaces are generous, controls are gentle, chips and avatars are fully round. No sharp 0px corners anywhere in the back office.

Components

Form inputs

Rounded controls with a green focus ring. Used sparingly — the trader surface is mostly read-only; inputs live in account setup, billing, and admin tools.

Components

Tabs & segmented control

A compact rounded segmented control for filtering tables and switching views. The active segment lifts onto an elevated surface.

Components

Alerts

Inline alerts borrow the regime palette. The critical exchange-disconnect alert escalates to a full-width red banner that drops between the top bar and content.

Components

Code blocks

For the SQL console, command runner, and audit log. Mono, rounded, on an elevated surface with green keywords and green numerics.

Components

Empty states

Calm, centered, never cute. A bordered icon, a terse line, and a single quiet action. Used on pages still in the build queue and for genuinely empty tables.

No open positions

The engine is flat. New positions resume automatically when the regime clears and a setup qualifies.
