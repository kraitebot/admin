Design system

The component foundation for the Kraite operator console. Dark chrome, calm light content, terminal-green accent, monospaced numbers. Trading semantics are sacred — green is long / profit / safe, red is short / loss / risk, never inverted. Surfaces use a soft radius scale; numbers are the hero.

Foundations

One signal color — terminal green — on a neutral ink scale. A fixed P&L pair and a five-step Black Swan regime palette carry all market meaning. No decorative gradients.

Components

Position tile

The hero working component. Direction governs the whole tile: green for longs, red for shorts — applied to the 2px border, the inverted ribbon header, the lifecycle track, the metric labels, and the accent values. The header band carries the market, side tag, leverage and ETA; the lifecycle track shows current price (PX) heading toward the take-profit target (TP) along a limit-order ladder; two metric rows below read the position's progress and prices. Every metric label has a ? affordance with a styled hover tooltip. An opening tile is desaturated until the position is established.

Motion

Motion is purposeful and quick — it confirms a state change, never decorates. Two signature movements: a single highlight that slides to the selected target (sidebar rail, segmented control) and a content crossfade on route change. Both decelerate into place with an expo-out curve and no overshoot.

Components

Form inputs

Rounded controls with a green focus ring. Used sparingly — the trader surface is mostly read-only; inputs live in account setup, billing, and admin tools.

Components

Tabs & segmented control

A compact rounded segmented control for filtering tables and switching views. The active segment carries a solid green pill that slides to the clicked option (420ms expo-out), matching the sidebar rail's motion.

Components

Alerts

Inline alerts borrow the regime palette. The critical exchange-disconnect alert escalates to a full-width red banner that drops between the top bar and content.

Components

Empty states

Calm, centered, never cute. A bordered icon, a terse line, and a single quiet action. Used on pages still in the build queue and for genuinely empty tables.
